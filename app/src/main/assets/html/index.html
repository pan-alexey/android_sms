Необходимы разрешения

Ввиду специфики работы приложения, необходимо предоставить все необходимые разрешения

  • textsmsОтправлять и просматривать SMS-сообщения

    Данное разрешение необходимо для отправки и получения СМС.

  • callОсуществление телефонных звонков и управления ими

    Данное разрешение необходимо для предоставления информации о радиомодуле телефона, информации по imei. Разрешение необходимо для отправки USSD запроса.
